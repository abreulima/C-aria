Printing the ASCII Table

As you told me before, we know the ASCII table contains 127 elements (some of them aren't printables), and every element is identified by a number.

For example, compiling the line printf("%c", 75); we'll see on the terminal the beautiful letter "K".

The %c inside the printf will convert an integer to the correspent symbol in the ASCII table, if we write %d instead of %c, we would see the number 75 on the terminal.

ASCII Table

From number 33 (!) to 126 (~), they're printables, it means we can see them on the terminal.

Our tasks is to print all the printables elements on the terminal using loops (while and for)!

It is simple task, but we need to understand how a loop works to do it without doubts.
